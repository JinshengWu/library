## 技术栈
 - 数据库：MySQL
 - 开发工具：IDEA
 - 数据连接池：Druid
 - Web容器：Apache Tomcat
 - 项目管理工具：Maven
 - 版本控制工具：Git
 - 后端技术：Spring + SpringMVC + MyBatis（SSM）
 - 前端框架：LayUI
